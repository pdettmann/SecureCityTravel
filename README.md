# SecureCityTravel

**Steps to run this project:**

1. clone the project

**Change directory into `backend`**

1.  `npm i` to install the npm-modules defined in package.json
2. create a ".env" file with valid 
    - CONNECTION_STRING,
    - API_PORT,
    - TOKEN_KEY,
    - CLIENT_URL
    - MAILGUNAPIKEY
3. use the command `npm run dev` to start the API locally
